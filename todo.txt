Rating service

Основные функции:
Подсчёт рейтингов политиков.
Обновление рейтингов при изменении статуса обещаний.
Предоставление рейтингов другим сервисам и пользователям.
Таблицы базы данных:
Rating
ID (uint64): Уникальный идентификатор рейтинга.
politicianID (uint64): Внешний ключ, ссылающийся на ID в таблице Politician.
rating (int): Рейтинг политика.
created_at (datetime): Дата и время создания записи.
updated_at (datetime): Дата и время последнего обновления записи.
API Endpoints
/ratings (GET)
Описание: Получение списка всех рейтингов.
Ответ: Список рейтингов с основной информацией (ID, politicianID, рейтинг).
/ratings/{politicianID} (GET)
Описание: Получение рейтинга конкретного политика.
Входные данные: ID политика.
Ответ: Рейтинг политика (ID, politicianID, рейтинг).
/ratings/{politicianID} (PUT)
Описание: Обновление рейтинга политика.
Входные данные: ID политика, новый рейтинг.
Ответ: Обновлённая информация о рейтинге политика.
Взаимодействие с другими сервисами (optional):
Politician Service:
При изменении информации о политике Rating Service обновляет рейтинги.
Rating Service предоставляет рейтинги для отображения на страницах политиков.
Promise Service:
При обновлении статуса обещания Promise Service уведомляет Rating Service, который пересчитывает рейтинг политика.
